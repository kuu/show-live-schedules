# show-live-schedules

## Install
```
$ git clone git@github.com:kuu/show-live-schedules.git
$ cd show-live-schedules
$ npm install
```

## Config
Put config file(s) in your work directory.
```js
 $ mkdir config
 $ vi config/default.json
 {
   "api": {
     "key":        "Your Ooyala API Key",
     "secret":     "Your Ooyala API Secret"
   }
 }
```

## Run
```
$ npm run show
```
